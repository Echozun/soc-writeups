# SOC Writeups

Room/box writeups and detection rules from building toward a Tier 1/2 SOC Analyst role.

- `writeups/SOC_Level_1/` — TryHackMe SOC Level 1 path rooms, grouped by module
- `writeups/HTB_Sherlocks/` — HackTheBox Sherlock forensic investigations, grouped by category (Network Forensics, Memory Forensics, etc.)
- `writeups/images/<slug>/` — screenshots referenced by each writeup, one folder per writeup
- `detections/` — Sigma/SPL rules, each tied to a specific writeup
